# PassGAN
PassGAN：使用GANs網路進行類人思維的密碼生成。

## 作者
**ChengChinHsia**

## 模型選項
- **1000萬**
- **3000萬**
- **5000萬**
- **1億**

## 資料集
PassGAN訓練所使用的綜合資料集總計9GB，包括：
- `rockyou`資料集
- 公開洩露的密碼
- 個人收集的密碼
- 電話號碼

## 貢獻
歡迎並鼓勵社群成員以提出問題、提交拉取請求或提供反饋等形式對PassGAN作出貢獻。
