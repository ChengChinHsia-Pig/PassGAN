# PassGAN
PassGAN:A Human-like Password Generation with GANs.

## Author
**ChengChinHsia**

## Model Options
- **10 million**
- **30 million**
- **50 million**
- **100 million**

## Dataset
PassGAN is trained on a comprehensive dataset totaling 9GB, which includes:
- The `rockyou` dataset
- Publicly leaked passwords
- Personally collected passwords
- Telephone numbers
